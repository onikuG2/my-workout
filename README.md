# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/16m0xGKfGC5Sg4951gPp2BDKqOkPltPiF

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Usage

このアプリケーションでは、オリジナルのワークアウトを作成し、実行することができます。

### 1. ワークアウトの作成

1.  **「新しいワークアウトを作成」** ボタンをクリックします。
2.  ワークアウトに名前を付けます (例: 全身HIIT、胸の日など)。
3.  **「エクササイズを追加」** ボタンを押し、トレーニング種目を追加します。
    *   **部位**と**種目**を選択します。
    *   **時間**または**回数**を設定します。
        *   時間を0にすると、回数ベースのトレーニングになります。
    *   **休憩時間**、**セット数**、**重量** をお好みで設定します。
4.  すべてのエクササイズを追加したら、**「保存」** ボタンをクリックします。

### 2. おすすめワークアウトの追加

メイン画面の **「おすすめワークアウトを追加」** ボタンから、あらかじめ用意されたワークアウトプランをリストに追加することができます。

### 3. ワークアウトの実行

1.  リストから実行したいワークアウトの **再生 (▶) ボタン** をクリックします。
2.  プレイヤー画面が開き、ワークアウトが開始されます。
    *   時間ベースの種目はタイマーが自動で進みます。
    *   回数ベースの種目は、画面中央のボタンを押すと次のセットまたは休憩に進みます。
3.  **「||」** ボタンで一時停止、**「▶」** ボタンで再開できます。
4.  **「«」** と **「»」** ボタンで、セットやエクササイズのスキップ/バックが可能です。
5.  **「ワークアウトを終了」** ボタンでいつでも中断できます。

### 4. ワークアウトの編集と削除

*   各ワークアウトの **鉛筆 (✎) ボタン** をクリックすると、内容を編集できます。
*   **ゴミ箱 (🗑️) ボタン** をクリックすると、ワークアウトを削除できます。

### 5. ワークアウト履歴の確認

1.  メイン画面右上の **「ワークアウト履歴」** ボタンをクリックします。
2.  過去に完了したワークアウトの一覧が表示されます。
3.  不要になった履歴は **ゴミ箱 (🗑️) ボタン** で削除できます。

### 6. データの管理

メイン画面下部の「データ管理」セクションでは、作成したワークアウトデータをファイルとして保存（エクスポート）したり、ファイルから読み込む（インポート）したりすることができます。これにより、データのバックアップや異なるデバイス間でのデータ移行が可能です。
