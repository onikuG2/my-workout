# コミットメッセージ

## 主要な機能追加・改善

### 1. ワークアウト作成機能の改善
- 「時間」を「ワークタイム」に文言変更
- ワークタイム、回数、休憩に任意入力機能を追加（プルダウンの「任意入力…」選択時に入力欄を表示）

### 2. 部位・種目マスター機能の実装
- ユーザーが独自の部位・種目を追加・削除できるマスター管理機能を追加
- マスターデータのエクスポート/インポート機能を実装
- ワークアウトデータとマスターデータを同時にエクスポート/インポート可能

### 3. ツイート機能の実装
- ワークアウト完了時と履歴からのツイート機能を追加
- テキストツイートにエクササイズ詳細を含める機能を実装
- 画像生成ツイート機能を追加（html2canvasを使用）
- 生成した画像をクリップボードにコピーしてツイートできる機能を実装
- ツイート内容にサービスURLとハッシュタグ（#マイワークアウト）を追加

### 4. ワークアウト履歴の改善
- 履歴エントリーにエクササイズ詳細情報を保存する機能を追加
- 履歴からの画像生成ツイート機能を実装

### 5. 音響効果の改善
- カウントダウン音をよりクリアで心地よい音に改善（ハーモニック追加、エンベロープ適用）
- ワークアウト完了時に華やかなメジャーコードアーペジオを再生する機能を追加

### 6. 依存関係の追加
- html2canvas を追加（画像生成機能用）
